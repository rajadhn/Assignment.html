Element and Tag in HTML

● What is a tag:-Tags are the building blocks of HTML documents and consist of angle brackets enclosing specific keywords. These tags provide instructions to web browsers on how to display the content.


●Types of Tags:-HTML tags can be broadly categorized into two types:

1.Paired Tags (or Container Tags) 2.Singular Tags (or Void tags)

What is an element

HTML elements consist of a start tag,an end tag(if applicable),and content in between the tags.

Grouping of HTML elements

You'll primarily use the div tag to group similar content together so you can style it easily. A great example of this is using div to group different sections of a webpage together. You can put together the header, nav, sections, and footer of a page in an individual div tag so they can be styled
